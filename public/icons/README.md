# PWA Icons

This directory contains PWA icons in various sizes. For now, placeholder icons are created. Replace these with properly designed icons for production.

Required icon sizes:
- 72x72
- 96x96
- 128x128
- 144x144
- 152x152
- 192x192
- 384x384
- 512x512

You can generate these from your profile image using tools like:
- Online: https://realfavicongenerator.net/
- CLI: `npx sharp input.jpg -o output.png --resize 192 192`
